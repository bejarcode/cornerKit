# CornerKit

> Pixel-perfect iOS-style squircle corners for the web

A lightweight (3.6KB gzipped), framework-agnostic JavaScript library for creating pixel-perfect iOS-style squircle corners using Figma's corner smoothing algorithm.

## Installation

```bash
npm install @cornerkit/core
```

See the [full documentation](packages/core/README.md) for complete usage instructions.

## Features

- **Tiny**: 3.6KB gzipped - 27% under 5KB budget
- **Fast**: <10ms render time, 60fps maintained during resize
- **Zero Dependencies**: Fully self-contained
- **TypeScript**: Full type definitions included
- **Universal**: Works in all modern browsers + IE11 fallback
