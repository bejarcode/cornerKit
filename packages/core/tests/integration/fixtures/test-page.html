CornerKit Figma Algorithm Testing

Compare cornerKit squircles with your Figma designs

üß™ How to Test

Open the browser console (Cmd+Opt+J on Mac, Ctrl+Shift+J on Windows) and try these commands:

// Compare smoothing values
ck.update('#ios-standard', { radius: 24, smoothing: 0.6 });  // iOS 7 (Figma standard)
ck.update('#figma-90', { radius: 24, smoothing: 0.9 });      // Figma 90%
ck.update('#old-default', { radius: 24, smoothing: 0.8 });   // Old default

// Test different sizes
ck.update('#test-square', { radius: 40, smoothing: 0.6 });
ck.update('#test-wide', { radius: 30, smoothing: 0.6 });
ck.update('#test-tall', { radius: 30, smoothing: 0.6 });

// Inspect the configuration
ck.inspect('#ios-standard');
